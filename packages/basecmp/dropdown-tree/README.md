## Props

| Name        | Type   | Default  | Description |
|:------------|:-------|:---------|:------------|
| data        | Object | - （必填）   | 以根节点开始的树    |
| value       | String | -        | 默认选中项       |
| placeholder | String | 请选择      | 提示语         |
