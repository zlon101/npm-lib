## Props

| Name  |    Type  | Default  | Description |
| :-----| :--------| :------- | :--- |
| value | Number   |     0    | 双向绑定的值 |
| min   | Number   |     0    | 最小值，需要>=0 |
| max   | Number   |     -    | 最大值，需要>=0 |
| unit  | String   |     -    | 单位 |

## 事件
| Name | Description|
| :--- | :---|
| input | 输入框发生更改时触发，返回当前输入框的内容 |
