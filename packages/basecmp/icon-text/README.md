##  参数

| Name       | Type          | Default | Description            |
|:-----------|:--------------|:--------|:-----------------------|
| icon       | String        | -       | 图标的css类名               |
| iconMargin | String        | 4       | 图标和文字的间距               |
| color      | String        | #06003b | 文字的颜色                  |
| fontSize   | String        | 14      | 文字的font-size           |
| to         | String、Object | -       | 链接的路由                  |
| target     | String        | -       | router-link 的 target 值 |

## 事件

无

## 插槽

| Name    | Description |
|:--------|:------------|
| default | 文本          |
| icon    | 自定义图标组件     |
