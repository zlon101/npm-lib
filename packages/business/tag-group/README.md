## Props

| Name  |    Type  | Default  | Describe |
| :-----| :--------| :------- | :--- |
| title |   String |  -  | 标签展示列表左侧标题 |
| tags  |   Array  |  -  | 标签列表数据 { id, label, icon } |
| initTips | String | 添加 | 标签列表占位文本 |

## 事件

| Name  | Describe |
| :-----| :--- |
| edit | 修改选择标签后触发 |
| delete | 点击标签删除icon后触发，返回删除tag的id |
