## Props

| Name      |    Type  | Default  | Describe |
| :-------- | :--------| :------- | :---     |
| data      | Array    |    []    | 数据内容 [{title: '', style: '', desc: ''}]，其中desc可以为String,Array,Object|
| title     | String   |          | 组件左侧标题 |

### Slot
| Name     | Describe |
| :------- | :------- |
| left     | 左侧内容 |
| mid      | 中间内容 |
| right    | 右侧内容  |
